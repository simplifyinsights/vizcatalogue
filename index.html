About

This was created by Toan Hoang, Tableau Zen Master as a way to quickly search for tutorials for the visualisations you require. This is an initial version, but more features will be coming.

Upcoming features include:

  • More In-depth descriptions for each data visualisation
  • When to use each data visualisation type
  • Filter Buttons for Categories
  • Alternatives

Visit website: https://tableau.toanhoang.com
